Unit 11: Model Selection and Evaluation

Completed

Welcome to Week 11.

This unit expands on model selection, evaluation, and deployment, which are critical processes in the machine learning (ML) workflow.

Selecting the correct model is akin to choosing the right tool for a given prediction or classification task. However, evaluating and monitoring model performance ensures reliability and effectiveness in real-world applications.

Additionally, this unit introduces MLOps (Machine Learning Operations), an essential framework for maintaining and deploying models in production environments. We will examine how automated model monitoring, retraining, and deployment pipelines enhance the robustness of ML systems.

This week we shall learn about:

  • The workflow of model selection, evaluation, and deployment.
  • Techniques for optimising hyperparameters to enhance performance.
  • Metrics beyond accuracy—choosing appropriate evaluation methods.
  • MLOps concepts—ensuring model reproducibility and scalability in production.

On completion of this week, you will be able to:

  • Understand the core advancements in NLP and how they are transforming industries.
  • Understand the importance of model selection, evaluation, and optimisation.
  • Apply appropriate evaluation metrics for different ML tasks.
  • Explore MLOps principles for continuous model improvement and monitoring.

ASSIGNMENT BRIEF

Neural Network Models for Object Recognition Using Multi-Track ML Approaches

📊 Assignment Overview

This assignment requires developing and evaluating an object recognition model using the CIFAR-10 dataset to compare different machine learning approaches and assess their effectiveness in automated visual recognition tasks.

🎯 Chosen Track: Track 1 - Classical ML (Feature Engineering + SVM/KNN) - Feature extraction, object classification without deep learning.
🔍 Track 1: Classical ML

Focus: (Feature Engineering + SVM/KNN).

Methods: Feature extraction, object classification without deep learning.

🧠 Track 2: Deep Learning

Focus: (CNNs & Transfer Learning).

Methods: Convolutional Neural Networks, data augmentation, fine-tuning pre-trained models.

🚀 Track 3: Advanced ML

Focus: (Self-Supervised Learning & Neural Architecture Search).

Methods: Contrastive learning, automated neural network optimisation.

📝 Assignment Tasks

Train and evaluate an object recognition model using their assigned approach.
Compare training, validation, and test performance.
Analyse the impact of hyperparameters, architecture, and data augmentation.
Discuss the advantages and trade-offs of their assigned ML approach.
Present their findings in a 20-minute recorded presentation.

Personal Reflection

Preparing this Track 1 classical ML assignment proved to be both challenging and immensely rewarding.

The technical complexity of implementing feature engineering from scratch - extracting HOG, LBP, and color features while managing dimensionality reduction through PCA - required significant time investment and debugging patience.

Balancing the theoretical understanding of SVM and KNN algorithms with practical implementation created stressful moments, particularly when troubleshooting code errors and optimizing hyperparameters.

The pressure of creating a professional 20-minute presentation that effectively communicated complex ML concepts added another layer of intensity.

However, witnessing the models successfully classify CIFAR-10 images and seeing authentic results emerge from carefully engineered features brought profound satisfaction.

"The journey from initial confusion to genuine understanding of classical ML principles, combined with developing both technical skills and presentation abilities, made every challenging moment worthwhile.

This assignment transformed theoretical knowledge into practical expertise, creating lasting confidence in machine learning fundamentals."
