Unit 2: Sets, Set Theory, Truth Tables and Logic.

Completed

Welcome to Week 2.

This unit introduces the underlying principles of reasoning: that is set theory, logic, and truth tables. It also provides a number of exercises to provide practice in manipulating sets and truth tables.

This week we shall learn about:

  • Review sets and truth tables.
  • Discuss how sets and truth tables are related to logic and reasoning.
  • Practice how to use sets and truth tables to resolve logic problems.

On completion of this week, you will be able to:

  • Describe the rules of set and truth table manipulation.
  • Use sets to represent logic relationships.
  • Use truth tables to resolve logic problems.
